How It Works

How the Sequence Prediction Program Works

This program predicts the next numbers in a given sequence, analyzes the pattern, and displays the confidence of the prediction. It can detect various types of mathematical progressions and use regression when the sequence doesn't fit a common pattern. Below is a detailed explanation of each part:

1. User Input

When you visit the web application, you’re prompted to enter 5 numbers (separated by spaces) representing the sequence you want to analyze. For example:

1 4 9 16 25

This input is then processed by the program to identify the pattern and make predictions.

2. Identifying the Sequence Type

The first thing the program does is analyze the sequence to determine if it follows any well-known pattern. Here’s how it works:

  • Arithmetic Progression (AP): The difference between consecutive terms is constant. For example: 2, 4, 6, 8, 10 (Difference of 2 between each term).
  • Geometric Progression (GP): The ratio between consecutive terms is constant. For example: 2, 4, 8, 16, 32 (Each term is doubled).
  • Fibonacci Sequence: Each term is the sum of the two previous terms. For example: 1, 1, 2, 3, 5 (1 + 1 = 2, 1 + 2 = 3, etc.).
  • Harmonic Progression (HP): The differences between the reciprocals (1/x) of the terms are constant. For example: 1, 1/2, 1/3, 1/4, 1/5 (Differences in reciprocals are equal).
  • Factorial Progression: Each term is the factorial of a number. For example: 1, 2, 6, 24, 120 (Factorials of 1, 2, 3, 4, and 5).

The program checks the differences or ratios between consecutive terms and matches the sequence to one of these types.

3. Predicting the Next Numbers

Once the program identifies the type of sequence, it calculates the next three numbers based on the identified pattern:

  • For Arithmetic Progression (AP): It adds the constant difference to the last number in the series.
  • For Geometric Progression (GP): It multiplies the last number by the constant ratio.
  • For Fibonacci Sequence: It adds the two previous terms to predict the next terms.
  • For Harmonic Progression (HP): It uses the common difference in the reciprocals to calculate the next terms.
  • For Factorial Progression: It computes the next factorial numbers.

4. When the Sequence is Not Recognized (Polynomial Regression)

If the program cannot identify the sequence type (for example, if the sequence is irregular), it falls back to Polynomial Regression. Here’s what happens:

  • Polynomial Regression fits a curve to the existing data using a second-degree polynomial (quadratic function).
  • The program then uses this fitted curve to predict the next three values.

This method works well for sequences that don't follow any simple pattern, as it tries to find the best-fitting curve for the data.

5. Confidence Calculation

For sequences where Polynomial Regression is used, the program calculates a confidence score to show how well the regression model fits the given data:

  • The R² score (Coefficient of Determination) is calculated, which indicates how well the regression line explains the data.
  • The program then converts this score into a percentage, representing the confidence level in the prediction.

This confidence score is displayed on the screen alongside the predicted next numbers.

6. Visualization of the Sequence and Predictions

To help you better understand the sequence and the predictions:

  • The program plots a graph showing:
    • The input series as blue dots.
    • The predicted next numbers as red dots.
    • A forecast line (if applicable) connecting the last number of the input series to the first predicted number.

The graph helps visualize how the predicted values fit into the pattern of the sequence.

7. Final Output

The program shows the predicted next 3 numbers, displays the confidence score (if Polynomial Regression was used), and generates a graph of the input sequence and predicted numbers.

The result could look like:

  • Predicted Numbers: [36, 49, 64]
  • Confidence: 95.2% (if using Polynomial Regression)
  • Graph: A plot with the series and predicted points.

How to Use the Application

1. Enter a sequence of 5 numbers in the provided input box.

2. Click the "Predict" button to receive the predicted next numbers.

3. The tool will:

  • Identify the sequence pattern (if possible).
  • Predict the next three numbers.
  • Display the confidence score and graph.
